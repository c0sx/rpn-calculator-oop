# rpn-calculator-oop

**консольная программа** вычисления значения произвольного математического выражения, содержащего следующие операции:

- унарные плюс и минус;
- сложение;
- вычитание;
- деление;
- умножение;
- скобки.

**Ввод:**

```(1 + 2) * 4 + 3```

**Вывод:**

``
1 2 + 4 × 3 +
``

``
15
``
